Project 1

NASA Rover ETL Pipeline

I combined my passion to explore data and engineering techniques in this Databricks ETL pipeline. The pipeline connects to the NASA Mars Rover API and extracts Mars Rover camera and photo data. I utilized a medallion architecture along with storing the data in AWS S3 locations forming a data lake. After cleaning and transforming the data, it is used to create various dashboards. One sample dashboard displays camera, mission and rover data. Another dashboard does image analysis on a photo extracted from the data lake. Technologies used were Databricks, Python, Pyspark, SQL, and AWS.

Project 2

BabyWatch Baby Monitor

I led a team of 3 engineers to create an AI-powered encrypted baby monitor. The project was deployed on a Jetson Nano using Python, Flask, OpenCV, and Sci-kit Learn. I wrote the Flask App, and the firmware for both the server and the camera on the Jetson Nano. I also built a custom data-set using a baby doll and trained various models with it. I pickled the best model and deployed it to the Jetson Nano, where it ran its machine learning algorithms in real time to notify a user if their child was in danger.

Project 3

YelpCamp

Yelpcamp is a campground review site built with Node.js, Express, and MongoDB. The full stack application features user authentication (sign up, login, logout), user profiles and campground ownership, campground ratings and reviews, and map integration. It also supports CRUD operations for campgrounds. This project is based off of Colt Steele's Web Developer 2024 bootcamp. You can check it out here: YelpCamp . Note that it might take up to a minute for the cluster to start if you're the first visitor in some time.
